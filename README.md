<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hoangviethung071195/training-vue_pokemon-game.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run at localhost`
   ```js
   npm run dev
   ```

<!-- USAGE EXAMPLES -->
## Usage

+ There are 4 game mode, you have to flip up and remember cards'position
+ If you flip two same cards, those cards will not flip down anymore
+ You will win if all cards are flipped up
